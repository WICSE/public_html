INTRO TO JAVASCRIPTING
JAVASCRIPTS ARE CLIENT-SIDE SCRIPTS THAT RUN BASED ON USER-INTERACTION WITH THE WEBPAGE.
YOUR HTML FILE NEEDS TO INCLUDE THE .JS IN THE HEADER.
ie.
script src="gitlogin.js" /script


script src="popup.js" /script

THERE ARE DIFFERENT WAYS TO RUN JAVASCRIPTS, SUCH AS:
1. FORMS
form method="post" action="cgi-bin/mail.php"
input type="text" name="name" size="60"
input type="text" name="email" size="40"
textarea name="message" cols="60" rows="10"/textarea

input type="submit" class="btn" value="Send" name="Submit"


2. LINKS

a href="javascript:doNothing()" onClick="login()">Protected Page /a

a href="javascript:doNothing()" onClick="popupHandle=popup()">Popup /a


NOTE: onTouch() is available for touch screens.

and onMouseOver() is available for regular browsers.

POPUPS

SAVE THIS CODE AS POPUP.JS
var popupHandle;
function popup(myPopup)
{
// always close the old one , so only one at a time is open
if(popupHandle || popupHandle!=null)
{
if (!popupHandle.closed) popupHandle.close();
}
popupHandle=null;

// open the popup
popupHandle = window.open( "popup.html", "myWindow",
"height = 500, width = 500, resizable = 0" )
return popupHandle;
}

function winclose()
{
if (window.popupHandle!=null && !window.popupHandle.closed)
{
window.popupHandle.close();
}
}

function doNothing(){} // does nothing but required by JavaScript in this case
