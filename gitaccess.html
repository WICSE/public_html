HOW TO USE THE WICSE GIT REPOSITORY


STEP 1:
INSTALL GIT ON THE COMPUTER YOU PLAN TO USE.

INSTALL LIBRARIES:

FEDORA-
YUM INSTALL CURL-DEVEL EXPAT-DEVEL GETTEXT-DEVEL \ OPENSSL-DEVEL ZLIB-DEVEL

DEBIAN-
APT-GET INSTALL LIBCURL4-GNUTLS-DEV LIBEXPAT1-DEV GETTEXT \ LIBZ-DEV

INSTALL GIT:

FEDORA-
YUM INSTALL GIT-CORE

DEBIAN-
APT-GET INSTALL GIT-CORE

MAC OR WINDOWS:
GET THE INSTALLER FROM GOOGLE CODE, EXECUTE

STEP 2:
ADD USERNAME AND PASSWORD FOR GIT

TYPE:
GIT CONFIG --GLOBAL USER.NAME "WICSE"
GIT CONFIG --GLOBAL USER.EMAIL WICSECLUB@GMAIL.COM


STEP 3:
CREATE AN SSH KEY FOR ACCESSING THE GIT REPOSITORY

1. TYPE SSH-KEYGEN INTO TERMINAL, THIS WILL GENERATE THE KEY
2. SAVE THE KEY IN /HOME/'YOURUSERNAME'/.SSH/ID_RSA
3. CHOOSE A PASSPHRASE (I DEFAULTED TO NONE)
4. CAT ID_RSA AND COPY KEYGEN TO CLIPBOARD


STEP 4:
LOGIN TO WICSE REPOSITORY ON GITHUB.COM
USERNAME: wicseclub@gmail.com
PASSWORD: 1qawsedrf
STEP 5:
ADD YOUR SSH KEY TO THE REPOSITORY

1. GO TO ACCOUNT SETTINGS
2. GO TO SSH PUBLIC KEYS
3. ADD ANOTHER PUBLIC KEY, COPY, PASTE AND CALL IT YOUR NAME PLUS YOUR COMPUTER'S NAME (YOU MAY WISH TO ADD YOUR HOME COMPUTER AND THE ECC LAB)

NOTE: YOUR ECC KEY WILL WORK FOR ALL ECC COMPUTERS BECAUSE IT IS LINKED TO YOUR ECC ACCOUNT


STEP 6:
CREATE A DIRECTORY ON YOUR COMPUTER TO HOLD WICSE PROJECTS

STEP 7:
OPEN A TERMINAL TO THIS DIRECTORY AND TYPE:
'GIT INIT'

THIS WILL INITIALIZE THE DIRECTORY AS A GIT REPOSITORY

STEP 8:
TYPE:
GIT REMOTE ADD ORIGIN GIT@GITHUB.COM:WICSE/'THE NAME OF THE REPOSITORY YOU WISH TO ADD' (i.e. PUBLIC_HTML)

THIS CREATES A LOCAL REPOSITORY FOR YOU ON YOUR COMPUTER THAT IS LINKED TO THE WICSE REPOSITORY
NOW YOU CAN USE GIT CHECKOUT TO COPY THE FILES FROM THE WICSE REPOSITORY TO YOUR LOCAL REPOSITORY.

USE GIT ADD TO ADD FILES

COMMIT USING GIT COMMIT -M 'MY MESSAGE'

IMPORTANT! AFTER COMMITING TO THE LOCAL REPOSITORY, CHANGES MUST BE 'PUSHED' TO THE SERVER USING GIT PUSH
